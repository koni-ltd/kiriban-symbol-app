# Kiriban Symbol App

「キリ番」をSymbolブロックチェーンで報告するアプリケーション。

## 概要

本アプリケーションは、Symbolブロックチェーン上の管理人宛送金トランザクションを取得し、表示するシステムです。

## 開発環境のセットアップ

### 前提条件

- Node.js (v18以上推奨)
- npm または yarn

### インストールと起動

1. リポジトリをクローンまたはダウンロードし、ディレクトリに移動します。
2. 依存パッケージをインストールします。

```bash
npm install
```

3. 開発サーバーを起動します。

```bash
npm run dev
```

4. ブラウザで指定されたローカルホスト（例: `http://localhost:5173/`）を開きます。
